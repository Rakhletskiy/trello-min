# Trello (min version)
Dashboard with drag'n'drop and local storage

## Features:
1. Add/del/edit cards
2. Add/del columns
3. Drag'N'Drop
4. LocalStorage

## Stack:
1. React
2. Redux(+redux-thunk)
4. AntDesign
5. React Beautiful DnD (3rd-party library)
6. SASS

Link: https://rakhletskiy.github.io/trello-min/
